This repository holds the results and analysis from a collection of experiments
testing out a new kind of genetic algorithm. These experiments explore
different *ways* of evolving Conway's Game of Life simulations that satisfy
various fitness goals, to see which technique produces the best results.

For more information on the motivation, design, and execution of the project
that produced these results, check out the [README file](https://github.com/ngaylinn/epigenetic-gol-prototype) for
the source repository.

To learn more about the experiments and their results, start with the [analysis
writeup](ANALYSIS.md), then feel free to explore the full experiment data
in this repository to your heart's content.
